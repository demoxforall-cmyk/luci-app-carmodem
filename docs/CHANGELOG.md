# CHANGELOG

История версий luci-app-carmodem. Версии — `PKG_VERSION` в `luci-app-carmodem/Makefile`.

## 3.4
- Status: в таблицах Carrier aggregation и Neighbour cells столбцы данных равной ширины (промежуток Type↔Band = остальным); ★-столбец не трогается.
- Messages: панель контактов сужена 38% → 34% (больше места переписке).
- Advanced / AT-консоль: добавлен реестр команд смены интерфейса RM520N — `AT+QCFG="usbnet"` (QMI/ECM/MBIM/RNDIS/NCM), `AT+QCFG="data_interface"` (USB/PCIe), `AT+CFUN=1,1`.

## 3.3
- Advanced / Connection: кнопка **живого журнала дозвона** — фрейм выше band-lock с логом ModemManager/netifd/wwan в реальном времени (бэкенд `get_conn_log`), повторное нажатие убирает.

## 3.2
- Бейдж «Connected · LTE» вынесен из `<h3>` (структурный фикс размера — типографика заголовка больше не раздувает пилюлю).

## 3.1
- Принудительная фиксация размера бейджа статуса в шапке Connection.

## 3.0
- **Один пакет**: русский перевод (`carmodem.ru.lmo`) вшит прямо в основной apk (`translations/carmodem.ru.po` → `po2lmo` в `root/`); отдельный `luci-i18n-carmodem-ru` больше не нужен.
- `uci-defaults` регистрирует «Русский» в System → Language.

## 2.9
- **i18n**: исходные строки переведены на английский, добавлен русский перевод; язык — по системному языку LuCI. Аббревиатуры (RSRP/SIM/USSD/…) остаются латиницей.

## 2.8
- Messages: убраны «галочки» ✓✓ (подтверждений нет) → показ времени; время — только у последнего в череде одинаковых; запоминание времени отправки.
- Messages: фикс «прыжка» ползунка при выделении сообщений к удалению.
- Status: выравнивание столбца Type в таблицах CA/Neighbour.
- Advanced / Connection: **живой статус** соединения (опрос раз в секунду, бэкенд `get_conn`).
- Убран лишний CBI-футер (Save & Apply / Save / Reset) на Advanced.

## 2.7
- Меню переименовано: **Status · Messages · Advanced** (порядок 1-2-3); заголовки страниц синхронизированы.

## 2.5–2.6
- **Advanced** полностью переоформлен в стиле «Console RF» (приборная панель): RAT-пилюли, SIM/TTL, band-lock LED-чипами + счётчик, поиск операторов (асинхронный), AT-консоль чёрным терминалом + реестр частых AT-команд RM520N.

## 2.4
- Advanced: активные band-lock отмечаются по факту (`get_bands`), поиск операторов переведён в асинхронный режим (фон + опрос), вёрстка SIM & TTL.

## 2.0–2.3
- **Messages**: SMS/USSD-мессенджер (контакты слева, переписка справа), 3 режима хранилища (SIM/Modem/Combined) с индикаторами заполнения, цветные аватары, разделители дат, удаление с выделением, новое сообщение на новый номер.
- Фиксы: корректное хранение/порядок сообщений (по id), кириллица в SMS (октал-декод mmcli → валидный JSON), тема-адаптивность (currentColor/color-mix), убрана секция Internet Status.

## 0.x–1.x
- Ядро бэкенда: AT-демон (сериализация `ttyUSB2/3`), rpcd-объект `carmodem`, парсеры (servingcell/CA/по-антенные/соседи/скан), телеметрия MM-first + AT для глубоких метрик.
- Базовые экраны LuCI (Modem Info / управление), цветовые пороги качества, N/A-рендеринг.
- Юнит-тесты на фикстурах реальных AT-ответов RM520N-GL.
